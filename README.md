# lindyhopseoul

A mobile-first single-page web app introducing Lindy Hop and the swing dance community in Seoul.
Built with React and Tailwind CSS, featuring multilingual support, responsive design, and a smooth flow from introduction to class registration.

## Tech Stack
- React
- Vite
- Tailwind CSS
- localStorage

## Run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
